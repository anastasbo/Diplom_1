---

## Проект:
- Библиотеки: JUnit 4, Mockito,  Jacoco;
- Amazon JAVA 11;

---

## Тесты:
- Создание бургера;
- Проверка булки бургера;
- Добавление ингредиента в бургер;
- Перемещение ингредиента в бургере;
- Удаление ингредиента в бургере;
- Проверка рецепта;
- Проверка цены;
- Проверка типа ингредиента.
